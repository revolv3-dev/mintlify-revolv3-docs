**EPX Test Cards**

EPX Credit Cards data for testing: Payment Account Number- 4000000000000002, Security Code - 123, Expiration Date - any but should be great than current date. Make amount 1.00 for success

EPX ACH data for testing: RoutingNumber - 031100092, AccountNumber - any [https://developer.paymentshub.com/supplemental-resources/epx-credit-card-test-data](https://developer.paymentshub.com/supplemental-resources/epx-credit-card-test-data)

Create a developer account to see various options on test cards or the most up to date.

To Trigger other responses use this information:

```
    "PaymentMethod": {
        "BillingAddress": {
            "AddressLine1": "123 N CENTRAL",
            "PostalCode": "12345"
        },
        "BillingFirstName": "Joe",
        "BillingLastName": "Smith",
        "CreditCard": {
            "PaymentAccountNumber": "4000000000000002",
            "ExpirationDate": "1225",
            "SecurityCode": "123"
        }
    },
```
